In the grandpa's notebook Nicola found the description of the old human game "Bulls and Cows". He wants to play with Stephan, but Nicola does not like to lose. And so we should help and write the program for this game (for guessing).

The host thinks 4-digits sequence. The digits must be all different. Then, in turn, you try to guess the sequence. If the matching digits are on their right positions, they are "bulls", if on different positions, they are "cows". The host says you how many "bulls" and "cows" in your guess. You should solve the code up to seven turns.

On each step your function receives a list with information from last steps. Each element of the list contains your guess and the result for it in the follow format "XXXX YBZC", where:

  • XXXX is a guess
  • Y is a quantity of "bulls"
  • XXXX is a quantity of "cows"

For each step your function returns your new guess as a string with four digits.

Let's consider an example. The hidden code is "5612". At the start your function receives the empty list. Your guess is "1234" -- two digits on the different positions but for the code. So the next call will be ["1234 0B2C"]. The next guess is "6512" -- two "bulls", two "cows". The next input is ["1234 0B2C", "6512 2B2C"]. And the last guess is "5612" and you win.

Input: A list with information from last steps.

Output: The guess as a string.

Example: 

checkio([])  # "1234"
checkio(["1234 0B2C"])  # "6512"
checkio(["1234 0B2C", "6512 2B2C"])  # "5612" and you win.

How it is used: This is a simple game that can amuse you (especially if play only with mind). But also your algorithm is an example of the prediction software.
